BMI Index Calculator

A lightweight tool for tracking health metrics and classification.

Overview

A simple yet effective lightweight tool that calculates Body Mass Index (BMI) based on user input and provides immediate health classification suggestions.

Key Features

  • Instant BMI calculation
  • Health category classification (Underweight, Normal, Overweight)
  • Weight goal recommendations
  • Clean and responsive user interface

Technologies Used

HTML5 CSS3 JavaScript

Purpose

To provide a quick way for wellness enthusiasts to track their health stats.
