DreamAI

AI-powered dream analysis and interpretation assistant.

Overview

DreamAI is an innovative application that interprets user dreams using advanced Natural Language Processing. It generates psychological and symbolic meanings to help users understand their subconscious mind.

Key Features

  • Dream text input analysis & interpretation
  • Conversation-style chatbot for deeper exploration
  • Voice-to-text dream recording
  • Dream history tracking and pattern recognition
  • Personalized dream warnings and insights

Technologies Used

NLP APIs (OpenAI / Gemini) Node.js / Python Backend HTML/CSS/JS Speech Recognition API

Purpose

To help users decode the meanings behind their dreams and track recurring subconscious patterns for better self-awareness.
