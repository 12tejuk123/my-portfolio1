Smart Waste Bin

IoT-enabled waste management automation.

Overview

A smart waste management system that utilizes sensors to monitor waste levels and automate disposal processes. It ensures timely collection and better hygiene in public spaces.

Key Features

  • Ultrasonic waste level detection
  • Automatic lid opening mechanism
  • IoT alert system (LEDs, Buzzer, SMS notifications)
  • Real-time status monitoring

Technologies Used

Arduino / NodeMCU Ultrasonic Sensors IoT (Blynk/MQTT) C++

Purpose

To automate smart city maintenance and maintain public cleanliness through efficient waste management.
