Archition

AI Architecture Design Assistant for layouts, blueprints, and cost estimation.

Overview

Archition is an AI–powered architecture assistant that helps architects and civil engineers design layouts, estimate material costs, and visualize 2D/3D structures instantly. It streamlines the preliminary design phase with intelligent automation.

Key Features

  • AI-based layout suggestions optimized for space
  • Auto–generated structural blueprints
  • Real-time cost & material estimation
  • Interactive 3D visualization using Three.js
  • Sustainable architecture recommendations
  • Export comprehensive plans as PDF

Technologies Used

Python (Backend AI) OpenAI / Gemini API Three.js HTML/CSS/JS Firebase

Purpose

To significantly reduce the time required for architectural planning and generate accurate, AI-backed design insights for professionals and students.
